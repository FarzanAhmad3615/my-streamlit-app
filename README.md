# Farzan InsightAI üß†‚ú®

**Farzan InsightAI** is a comprehensive, AI-driven educational ecosystem designed to simplify learning. By leveraging the **Gemini API** via the **Antigravity AI** Python wrapper, the platform transforms complex academic topics into digestible summaries, interactive quizzes, and personalized study aids.

---

## üöÄ Features

### 1. AI Topic Explainer

Break down any concept (e.g., Quantum Physics, Inflation, Portfolio Optimization) into three distinct difficulty levels:

* **Very Easy:** Simple analogies and core basics.
* **College Level:** Detailed and academic.
* **Technical:** In-depth, complex, and formula-heavy.

### 2. Notes & PDF Summarizer

Upload long lecture notes or technical PDFs. The system extracts:

* **Summary Points:** High-level takeaways.
* **Key Definitions:** Automated glossary generation (e.g., PD, LGD, EAD for Finance).

### 3. AI Quiz Generator

Test your knowledge with instant, AI-generated multiple-choice questions.

* Includes real-time scoring.
* Provides detailed explanations for every answer to reinforce learning.

### 4. Flashcard Generator

Uses **Active Recall** to turn topics into digital flashcards. Perfect for memorizing terms in NLP, SQL, or Medicine.

### 5. Learning Dashboard

Track your progress through a personalized dashboard showing:

* Average Quiz Scores.
* Recent Activity History.
* Focus Areas based on performance.

## üì¶ Installation

1. **Clone the repository:**
```bash
git clone https://github.com/yourusername/farzan-insightai.git
cd farzan-insightai

```


2. **Install dependencies:**
```bash
pip install -r requirements.txt

```


3. **Set up your API Key:**
Create a `.env` file in the root directory:
```env
GEMINI_API_KEY=your_api_key_here

```


4. **Run the application:**
```bash
python app.py

```
